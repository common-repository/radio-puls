This plugin places the radio player to your site.


== Description ==


Using this plugin you can add radio player on your site in few seconds, you can add your favorite radio stations and enjoy music with Radio Puls

= NOTE = 


Currently, you need a beta-invite for Radio Plugin to actually test the plugin.

== Installation ==


1. Upload the plugin zipfile to your Wordpress installation

1. Activate the plugin through the 'Plugins' menu in WordPress

1. Open your site and enjoy radio with Radio Puls Plugin
